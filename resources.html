## Physical books I own a physical copy of the following books and would be happy to lend them: * [Clean code](https://www.goodreads.com/book/show/3735293-clean-code) * A book about programming fundamentals - how to write better code. * Recommended background: none, content is accessible at any level. * [Object thinking](https://www.goodreads.com/book/show/43940.Object_Thinking) * A book about the philosophy behind object-oriented programing. * Recommended background: having modelled a domain using objects. * [Domain-Driven Design: Tackling Complexity in the Heart of Software](https://www.goodreads.com/book/show/179133.Domain_Driven_Design) * A book about object-oriented modelling of software systems. Quite practical. * Recommended background: having modelled a domain using objects. * [Release It!: Design and Deploy Production-Ready Software](https://www.goodreads.com/book/show/1069827.Release_It_) * A book about the mindset around building software that is stable, scalable, and easy to operate * Recommended background: having built a software system that was actually used by someone. * [Data & Reality: A Timeless Perspective on Perceiving and Managing Information in Our Imprecise World](https://www.goodreads.com/book/show/19781410-data-reality) * A book about philosophy and challenges of modelling the real world in data and software. * Recommended background: having defined a database schema. ## Online resources A few select resources I recommend exploring: * [The New Java Developer](https://javadevguy.wordpress.com/) blog * A great resource about fondamental ideas of object-oriented programming and how to apply them in Java code * I often refer to this blog when mentoring younger software developers * [The Morning Paper](https://blog.acolyer.org/) blog * A blog that summarizes/reviews various fresh and classic computer science research papers * I often recommend it to students as an accessible resource to get familiar with the mindset around scientific research, the structure of papers, and also to get a feeling of where the latest scientific research is focused, and also to discover areas they are curious about.
